## レコードフォーマット

### 共通仕様

- 全角文字: Shift JIS (2バイト)
- 半角文字(英・数・半角ｶﾅ): JIS8 (1バイト)
- レコード収録順序に仕様なし（事前告知なく変わる可能性あり）
- キー項目やデータ作成日を利用してDB反映すること

#### 初期値の定義

| 記号 | 意味 |
|------|------|
| `0` | 全エリアに半角数字 "0" をセット |
| `sp` | 全エリアに半角スペース " " をセット |
| `Ｓ` | 全エリアに全角スペース "　" をセット |
| `Ｓ sp` | 全角及び半角のスペースをセット(混在) |

## 特記事項

### 全般

- 1994年7月以前のデータには未整備項目が多数あり
- データ提供元の誤りは原則として訂正しない
- JRA以外(地方・海外)では設定されない項目・存在しないレコードが多数

### 2. レース詳細

- 中央競馬: 1986年以降を提供
- 地方競馬: 2000年以降の中央交流競走等
- 海外: 2000年以降の中央馬出走レース
- 前3/4ハロン、後3/4ハロン、コーナー通過順位、レコード更新区分: 2002年6月15日以降設定
- 海外レースでキーが重複する場合、レース番号が90番台になることがある

### 3. 馬毎レース情報

- データ区分1(出走馬名表)では馬番が初期値、データ区分2(出馬表)で有効値
- 出走馬名表と出馬表で内容が異なる場合がある
- 馬齢: 2000年以前は数え年、2001年以降は満年齢
- 後3ハロン: 基本的に後3ハロンのみ設定(後4ハロンは初期値)。障害は1F平均タイム

#### 脚質判定ロジック

- **逃げ**: 最終コーナー以外のいずれかのコーナーを1位で通過
- **先行**: 逃げ以外で最終コーナーを4位以内で通過
- **差し**: 逃げ・先行以外で最終コーナー通過順位が出走頭数の2/3以内(8頭未満は該当なし)
- **追込**: 上記以外

直線コースの場合は (走破タイム - 後3ハロンタイム) の値で判定。

### 13. 競走馬マスタ

- 収得賞金: クラス分けの基準となる賞金額
- 平成31年度夏競馬以降、降級処理廃止
- 着回数: 1986年以降のみカウント

### 競走条件コードの呼称変更 (平成31年度夏競馬以降)

|--------|--------|----------|

### グレードコード (2007-2009年)

- 国際格付けを持つ競走: 「G」表記
- それ以外: 「Jpn」表記
- JVData仕様では判別不可。別途CSVで対応: `http://dl.cdn.jra-van.ne.jp/datalab/grade/International.csv`

### 開催中止時の運用

- レース詳細・馬毎レース情報: データ区分9を提供
- 払戻: 提供なし
- 票数・オッズ: 発売後のみデータ区分9を提供
