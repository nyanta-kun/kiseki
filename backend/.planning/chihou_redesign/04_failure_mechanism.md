# 地方競馬 v10 指数 予測誤差メカニズム診断

対象: chihou.calculated_indices version=10 (LGB0.3+v9_0.7), 期間 2025-01〜2026-06, head_count≥6, ばんえい(course=83)除外。
**190,394 馬 / 18,421 レース**。勝ち馬 18,440 頭。

基準値: **指数1位 勝率33.4% / 複勝66.3%** vs **市場1番人気 勝率45.5% / 複勝76.9%**。

---

## ① 症状A: 人気馬(真の有力馬)を指数が下位に落とす

勝ち馬の指数rank分布: rank1 **33.4%** / rank2-3 32.2% / rank4-6 22.3% / rank7+ 12.0%。
→ **勝ち馬の34.4%を指数がrank4+に降格**(これが取りこぼしの本体)。

| 層別軸 | 取りこぼし(勝ち馬→rank4+)が多い条件 | demote率 |
|---|---|---|
| **過去走数(prior_runs)** | **21+走 50.9%** / 11-20走 40.5% / 6-10走 29.9% / **1-2走 18.9%(最良)** / debut 37.4% | 多走馬で激増 |
| **年齢** | **6歳+ 43.4%** / 5歳 41.0% / 4歳 28.9% / 3歳 30.7% / 2歳 30.5% | 高齢で増加 |
| **頭数** | 14+頭 40.9% / 12-13頭 39.2% / 10-11頭 34.3% / 6-7頭 24.1% | 多頭数で増加 |
| **コース** | 浦和39.0 / 船橋38.6 / 水沢38.2 / 名古屋36.4 が高、佐賀30.8 / 門別30.3 が低 | コース差は小さい(±4pt) |
| **距離** | 1901m+ 42.0% が突出。他帯は32-35%でほぼ平坦 | 長距離のみ悪化 |
| **サーフェス** | 芝44.0%(n=25・サンプル僅少) / ダ34.3% | 地方はダ主体で実質無影響 |

**主因**: 取りこぼしは「**多走・高齢・多頭数のベテラン馬**」に集中。過去走数21+で半数を落とす。
prior_runs 1-2走の馬(18.9%)が最も正確で、走数が増えるほど劣化する。
これは「直近フォーム/能力上昇を指数が拾えず、過去成績の平均に引きずられている」ことを示唆。
勝ち馬の市場人気で見ると pop1の13.3% / pop2の29.6% / pop3の46.4% を落としており、
**市場が2-3番人気に評価した勝ち馬を半数近く指数が捨てている**(=症状A本体)。

---

## ② 症状B: 凡走馬を指数が上位に過大評価

指数rank1-3の **bust率(fp≥6で凡走) = 25.7%**。市場top3のbust率20.3%より **+5.4pt 悪い**。

| 層別軸 | bust(指数top3→fp≥6)が多い条件 | bust率 |
|---|---|---|
| 過去走数 | 21+走 33.7% / 11-20走 28.0% / 1-2走18.4%(最良) | 多走で悪化(症状Aと同根) |
| 市場人気 | 指数top3でもpop6-8 54.4% / pop9+ 73.2% が凡走 | 人気薄を誤上位 |
| 年齢 | 5歳28.5% / 6歳+ 27.3% / 2歳20.7% | 高齢で悪化 |

**どのサブ指数が主因か(過大評価の犯人特定)**:
- 馬ごとにレース内z化し、**最も高いサブ指数(dominant)** 別に top3 bust率を見ると:

| dominant サブ指数 | n | bust率 | 勝率 |
|---|---|---|---|
| **last_margin_index** | 10,535 | **30.5%** | 16.6% |
| **rotation_index** | 9,246 | **28.2%** | 20.9% |
| speed_index | 11,069 | 24.4% | 24.2% |
| jockey_index | 16,278 | 23.8% | 24.4% |
| last3f_index | 8,134 | 22.4% | 21.8% |

→ **last_margin_index と rotation_index が dominant な馬で誤上位(高bust・低勝率)が起きる**。
speed/last3f/jockey が主導の馬は比較的健全。

**危険プロファイル(speed_z<0 だが jockey_z>0.8 or last_margin_z>0.8)**:
n=10,086、**bust 33.2% / 勝率15.2%**(top3平均 bust25.7%)。
→ 「能力(speed)は低いのに、着差・ローテ・騎手の見かけ値で上位に押し上げられた馬」が典型的ノイズ上位。

## ③ サブ指数の安定性(履歴の深さ別 予測精度)

合議の `composite_index` の **rank-IC(-spearman(composite, 着順))** を履歴深さ別に測定。
比較として市場 `pop` の rank-IC も併記:

| prior_runs | n | composite rank-IC | **市場pop rank-IC** | 指数1位 勝率 | win_prob較正(予測/実績) |
|---|---|---|---|---|---|
| 0 (debut) | 6,933 | **0.295** | **0.556** | 33.6% | 13.5% / 18.0%(過小) |
| 1-2 | 13,241 | 0.518 | 0.639 | 43.3% | 19.3% / 18.0% |
| 3-5 | 19,706 | 0.492 | 0.626 | 39.5% | 14.9% / 14.7% |
| 6-10 | 33,603 | 0.479 | 0.612 | 36.2% | 11.9% / 12.4% |
| 11-20 | 60,102 | 0.423 | 0.573 | 27.8% | 8.2% / 8.4% |
| **21+** | 56,809 | **0.387** | **0.552** | **20.0%** | 5.0% / 4.8% |

**所見**:
- **市場は全ての履歴帯で指数を圧倒**(pop rank-IC 0.55〜0.64 vs composite 0.29〜0.52)。指数は市場より一貫して劣る能力推定量。
- **debut(0走)が最悪**: composite rank-IC 0.295(市場0.556の約半分)。新馬は過去走特徴量が空でサブ指数が機能せず、ほぼランダム上位化。win_probも過小較正(13.5%予測/18.0%実績)。
- **21+走も劣化**: rank-IC 0.387・指数1位勝率20.0%。多走馬は古い凡走が平均を汚し、近走の好調を反映できない。
- 較正(win_probability)自体は概ね合っている(予測≈実績)→ 問題は**較正でなく順位付け(ランキング)能力**。
- 1-2走帯がスイートスポット(rank-IC 0.518・指数1位勝率43.3%で市場に最接近)。転厩・初コース判定カラム(prev_jockey_code等)は現状未使用。

## ④ 市場との差分の内訳 と 直すべき優先箇所

「指数1位 勝率33.4% vs 市場45.5%(差12.1pt)」を分解。

- 指数1位が **市場1番人気と一致(51.5%, n=9,478)** → 勝率 **50.1%**(市場以上)。
- 指数1位が **市場と不一致(48.5%, n=8,943)** → 勝率 **わずか15.8%**。
  - この不一致レースで、指数が見捨てた pop1 馬は **勝率40.5%**(指数rank 平均3.41位、66.1%をrank2-3、33.9%をrank4+に降格)。
  - 不一致時の指数1位の馬券構成: オッズ<3で勝率30.3%だが、**3-5倍18.4% / 5-10倍11.0% / 10-20倍5.9% / 20倍+2.6%** と、中〜高オッズの取りに行った馬がほぼ全滅。

**差分の定量化(反実仮想)**:
不一致レースで指数1位を捨てpop1に従えば、その帯の勝率は15.8% → 40.3%。
全体では指数1位勝率 33.4% → **45.3%(市場45.5%にほぼ一致)**。
→ **12.1ptの差のほぼ全量が「不一致レースでの誤判断」由来**。指数は一致時は市場以上だが、独自判断(disagreement)で踏み外す。

### 直すべき優先箇所(インパクト順)

1. **【最優先】不一致レースの順位付け劣化**: 市場pop1をrank4+に落とす33.9%のケースを是正するだけで大半が回復。
   能力推定がベテラン/debut で崩れる(③)のが直接原因。市場順位を弱い事前分布(prior)として明示的に取り込む、
   または market-blend を強化するのが最短。ただし「edge=不人気の妙味」を完全に潰さないよう、
   不一致を許す対象を「高rank-IC帯(1-5走・若駒)」に限定する設計が有効。
2. **【次点】last_margin_index / rotation_index の過大評価抑制(症状B)**: speed_z<0 ×これらが高い馬の誤上位(bust33.2%)。
   これらサブ指数の重みを下げる、または speed とのゲーティング(speed低なら margin/rotation を割引)で雑音上位を削減。
3. **【履歴対応】debut(rank-IC0.295)と21+走(0.387)の専用処理**: debut は血統/厩舎/騎手prior主体の別ロジック、
   多走馬は「近走 N 走のみ重み付け / 衰え検知」で古い凡走の希釈を防ぐ。prior_runs を特徴/層別に明示導入。
4. 多頭数(12+頭)・長距離(1900m+)は demote が増えるが、上記1-3の副次効果で改善見込み。コース差は小さく個別対応不要。

### 200字要約

v10指数1位の勝率33.4%は市場1番人気45.5%に12pt劣る。差の全量は「指数が市場と不一致のレース(48.5%)」由来で、ここで指数1位は勝率15.8%、捨てたpop1は40.5%勝つ。市場に従えば45.3%に回復。症状Aの取りこぼしは多走(21+走で50.9%)・高齢・多頭数のベテランに集中、症状Bの誤上位はlast_margin_index/rotation_index主導かつspeed低い馬(bust33%)が犯人。rank-ICは市場0.55-0.64に対し指数0.29-0.52でdebut(0.295)が最悪。較正は妥当で問題は順位付け。市場順位を事前分布化し不人気妙味は高IC帯に限定、margin/rotationをspeedでゲートするのが優先策。
