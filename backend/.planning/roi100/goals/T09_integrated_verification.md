# T09: 統合検証・ROI100% 判定ゲート【Wave 2・T02〜T06 依存】

## 目的
Wave 1 の部品を組み合わせた戦略ポートフォリオを T05 ハーネスで統一検証し、
本番化（T10)してよい戦略だけを選別する。**ここが ROI 100% の合否判定地点**。

## 判定基準（PLAN.md 準拠）
- 真OOS（2025-07〜2026-03）と純フォワード（2026-04〜）の**両方**で ROI ≥ 1.0、ブートストラップCI下限 ≥ 0.85
- 月あたりベット数 ≥ 10（運用可能性）
- 既存本番戦略（sweet_spot 1.188 / ワイド軸 1.05）を**下回る新戦略は追加しない**
